Item Name: The Item Name is visible to the public. Keep it generic (e.g., "Black Wallet", "Silver Laptop") so that fraudsters cannot use specific details to make false claims, this is one of the reasons we also decided to make the description private, instead of public anymore like stated in phase 1, because that would give away information.

Images: You are responsible for masking all sensitive data before uploading images. If uploading the image will impede the claiming process, leave the image empty, the system will render a placeholder image. Do not expose full student numbers, ID numbers, financial details, or identifying personal information.

Descriptions: The item description is completely private and will not be visible to the public. Please be as detailed and verbose as possible! Include specific identifying marks, serial numbers, hidden contents (like IDs or cash), and exact colors. The internal search engine scans these descriptions, so more detail means it will be easier to find when a student comes looking for it.
